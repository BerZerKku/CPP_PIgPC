# Настройка

## Настройка ОС

### Настройка ОС Windows

- Добавить переменную среды **AVR_FIND_ROOT_PATH** с указанием пути до toolchain avr. Например:
```
C:\MyProgram\avr8-gnu-toolchain-win32_x86
```
- Может потребоваться установка nmake.
